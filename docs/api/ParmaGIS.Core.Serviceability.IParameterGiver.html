Interface IParameterGiver

Namespace
ParmaGIS.Core.Serviceability
Assembly
ParmaGIS.Core.dll

Представляет интерфейс взаимодействия с сервисом получения параметров. Данный сервис позволяет зарегистрировать механизм разрешения зависимостей различных типав.

public interface IParameterGiver

Examples

Есть класс A, B и AParameterGiver:

class A { public int Property { get; set; } }

class B { public B(A a) { A = a; } }

class AParameterGiver
{
    public ParameterType => typrof(A);
    public object GetParameter(Type needy, ParameterInfo parameterInfo)
    {
        return new A();
    }
}

Класс А в данном случае предоставляется сервисом AParameterGiver. Таким образом, при разрешении зависимосетей для создания экземпляра класс B через Resolve(Type), классу В установится экземпляр класса А.
