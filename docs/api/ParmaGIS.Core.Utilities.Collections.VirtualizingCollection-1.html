Class VirtualizingCollection<T>

Namespace
ParmaGIS.Core.Utilities.Collections
Assembly
ParmaGIS.Core.dll

Специализированная реализация списка, которая обеспечивает виртуализацию данных. Коллекция делится на страницы, и страницы динамически извлекаются из IIItemsProvider при необходимости. Устаревшие страницы удаляются по истечении настраиваемого периода времени. Предназначено для использования с большими коллекциями в сети или на диске, которые не могут быть созданы локально из-за потребления памяти или задержки выборки.

Remarks

Реализация списка не полностью завершена, но ее должно быть достаточно для использования в качестве коллекции данных, привязанной только для чтения, к подходящему элементу управления ItemsControl.
