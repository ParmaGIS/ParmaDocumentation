Interface IModule

Namespace
ParmaGIS.Core.Modularity

ParmaGIS.Core.dll

Предаставляет атомарную часть механизма модулей. Реализация этого интерфейса в библиотеке позволяет определить ммдуль, зарешистрировать сервисы и различные дополнения. Не рекомендуется создавать в библиотеке больше одной реализации IModule.

public interface IModule

Methods

OnInitialized(IServiceProvider)

Инициализирует модуль. Метод вызывается после того как у всех модулей произойдет RegisterTypes(IServiceCollection). Гарантируется, что сервисы присутствующих модулей буду зарегистрированы до вызова этого метода.

serviceProvider IServiceProvider

Провайдер серивисов. Позволяет осуществить различные действия с сервисами.

RegisterTypes(IServiceCollection)

Позволяет зарегистрировать типы сервисов. Вызывается после сбора всех модулей. Определенная очередность вызова этого метода не гарантируется.
