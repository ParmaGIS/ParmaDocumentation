Представляет контракт взаимодействия с пространством данных. Пространство данных или просто пространство - центральное понятие в системе данных ParmaGIS. Пространство определяет то как данные из их источника представляются в экземплярах типов C#. Существуют различные пространства данных, которые отвечают за определенную часть системы данных источника. Источником данных является база данных, к которой определяется подключения через соответствующий провайдер, однако, в теории источником данных может быть что угодно. На данный момент не реализована какая-либо система валидации провайдеров для создания пространств, это возложено на плечи разработчиков пространства и строителя к нему.
